# けーしジャンプ

このフォルダには、LINEで共有できるシンプルなWebゲームが含まれています。

## 使い方

1. `index.html` をブラウザで開く
2. スマホでもプレイ可能
3. タップでジャンプして障害物を避けるゲームです

## 公開方法

### 1. Netlifyで公開する方法
1. netlify.com にログインまたはアカウント作成
2. 「New site from Git」または「Deploy manually」で公開
3. `index.html`, `style.css`, `script.js`, `けーし.png`, `けーし2.png` などをアップロード

### 2. GitHub Pagesで公開する方法
1. GitHubにリポジトリを作成
2. このフォルダをリポジトリに追加してコミット
3. GitHub Pages を有効化

### 3. 友達にLINEで送る方法
- 公開URLをLINEで送る
- ゲーム画面のスクリーンショットを一緒に添えると伝わりやすい
